Немного Википедии

«Монополия» — экономическая и стратегическая настольная игра для двух и более человек. Получила большую популярность в ХХ веке во многих странах мира. Цель игры — рационально используя стартовый капитал остаться единственным игроком, который не достиг банкротства. Фактически «Монополия» представляет собой игровое поле, состоящее из квадратов, которые проходят по кругу все игроки по очереди.
